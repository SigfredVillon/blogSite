# blogSite
 A blog site named Open forum where the user can create update delete his own post. As well as like and comments other's post.


HOW TO SETUP
1. Make Sure Composer and Laravel is installed 
2. Put this folder into the htdocs of your XAMPP
3. Change the database details into your mysql details in the .env file
4. Run the migrations
5. Run "php artisan serve" and "npm run dev" 
6. Open the link
7. Now you are done! You may now explore the website


Versions
PHP 8.2.4 
Laravel Framework 11.13.0
